# 🛡️ SKAntiBot
**Підтримувана версія гри:** `1.16.5`

---

### 📖 Про плагін
**SKAntiBot** — це легке та ефективне рішення для захисту вашого Minecraft-сервера від автоматизованих бот-атак.

Замість застарілих та нудних текстових команд, плагін пропонує гравцям пройти інтерактивну капчу через графічний інтерфейс (GUI). Такий підхід робить обхід захисту практично неможливим для більшості сучасних бот-програм, забезпечуючи при цьому комфортний вхід для реальних користувачів.

---

### ✨ Ключові особливості

* **🖱️ Interactive GUI Captcha**
  Замість введення тексту в чат, гравець має взаємодіяти з предметами в меню. Це значно підвищує рівень безпеки, оскільки боти рідко вміють розпізнавати об'єкти в GUI.
* **🎲 Система рандомізації**
  Кожного разу предмети та їх розташування в меню перемішуються. Це унеможливлює створення простих макросів або скриптів, налаштованих на клік по конкретному слоту.
* **⚡ Висока продуктивність**
  Код плагіна максимально оптимізований: він не створює навантаження на сервер та коректно очищає сесії гравців після їх виходу, запобігаючи витоку пам'яті.
* **🧹 Автоматичне очищення сесій**
  Плагін інтелектуально керує даними підключень, що дозволяє стабільно працювати навіть під час масованих атак.
